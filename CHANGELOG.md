# 변경 기록

## 1.0.1 - 2026-07-24

### 문서

- 통합 게임 기획서를 실제 4라운드·시간의 방·최종 발의·랭킹 흐름에 맞춰 현행화
- 상세 기능 설계서를 현재 중앙 팝업 화면과 저장·복구 동작 기준으로 재작성
- 코딩 설계서에서 폐기된 Classic ASP·MSSQL 구조를 제거하고 Puter.js·Firebase·localStorage 구조로 교체
- 사건 콘텐츠 기획서에 현재 단서, 전체 정답, 결말과 이미지 자산을 반영

## 1.0.0 - 2026-07-23

### 추가

- 네 라운드로 진행되는 CASE 01 `마크의 마지막 수리`
- 현장 사진별 단서 3개 조사
- 라운드별 시간의 방 추가정보 3개
- Puter.js 기반 AI 동료 의견과 추천 질문
- 사실표 기반 AI 집사 질문 판정
- 개인 시간, 질문 횟수와 시간의 방 비용
- 최종 발의 여섯 항목과 정답·부분정답·오답 평가
- 네 장면으로 설명하는 사건의 전체 내막
- 브라우저 진행 저장과 새로고침 복구
- Firebase Google 로그인과 공개 랭킹
- 모바일 중앙 팝업형 진행 UI
- 개발자 정보 페이지와 공유 기능
- 전체 공개용 README, 테마 제작 가이드와 기여 문서
