# Threads 공개 글 초안

피의게임 X를 보고 팬의 마음으로 AI 미스터리 팬게임을 만들어봤습니다.

이름은 **마크의 마지막 수리**.

사진 속 단서를 찾고, 동료들의 의견을 듣고, 집사에게 질문하면서 사건의 전말을 맞히는 게임입니다. 질문하거나 시간의 방에서 힌트를 얻을수록 내 시간이 늘어나기 때문에 추리뿐 아니라 시간 관리도 필요합니다.

기획부터 코딩, 디버깅, 게임 이미지까지 GPT와 대화하는 바이브코딩으로 만들었습니다.

그리고 이 게임의 개발 소스를 전부 공개합니다.

코드를 잘 몰라도 공개 소스를 Fork한 뒤 AI에게 새로운 사건을 설명해서 자신만의 미스터리 테마를 만들어보세요. 사건 데이터, AI 판정 규칙, Firebase 랭킹과 모바일 UI까지 함께 공개합니다.
