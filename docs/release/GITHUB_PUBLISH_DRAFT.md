본문:

```markdown
피의게임 X를 보고 팬의 마음으로 만든 비공식 AI 미스터리 팬 프로젝트를 공개합니다.

이 게임은 기획, 코드, 디버깅과 이미지 제작까지 GPT와 대화하는 바이브코딩 방식으로 완성했습니다. 사진 속 단서를 조사하고 AI 동료의 의견을 들은 뒤, 집사에게 질문하며 네 라운드에 걸쳐 사건의 전말을 추리합니다.

### 포함된 기능

- 네 라운드 사진 중심 사건 전개
- 현재 공개된 정보만 사용하는 AI 동료
- 예·아니요·그럴 수도 있습니다·중요하지 않습니다 AI 판정
- 질문과 시간의 방 사용에 따른 개인 시간 증가
- 여섯 항목의 최종 발의와 항목별 평가
- 모바일 중앙 팝업 UI
- 로컬 진행 저장과 Firebase Google 로그인 랭킹

### 전체 개발 소스 공개

게임에 사용한 HTML, CSS, JavaScript, AI 프롬프트, 사건 데이터, Firebase 규칙과 선택형 Classic ASP 서버 소스를 공개합니다.

이 소스를 Fork하고 `docs/guides/CUSTOM_THEME_GUIDE.md`를 따라 새로운 사건과 테마를 바이브코딩으로 만들어 보세요.
```
