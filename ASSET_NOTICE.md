# 이미지와 아이콘 사용 안내

## 게임 장면 이미지

이 프로젝트의 배전실, 라운드, 결말 장면 이미지는 GPT 이미지 생성과 후처리를 통해 제작했습니다. 새로운 미스터리 테마를 만들 때 참고하거나 교체할 수 있습니다.

## 동료 초상화

동료 초상화는 게임 속 가상 인물을 표현하기 위한 에셋입니다. 다른 프로젝트에서는 실제 인물로 오인되지 않도록 새로운 가상 인물 이미지로 교체하는 것을 권장합니다.

## 외부 아이콘

- GitHub 로고는 GitHub의 브랜드 가이드 적용을 받습니다.
- Threads 아이콘은 Simple Icons의 CC0 제공 에셋을 바탕으로 사용했습니다.
- `SON 키우기 타이쿤 : 아들을 축구 월클선수로` 아이콘과 명칭은 Water Castle Games의 별도 게임 홍보 자산입니다.
